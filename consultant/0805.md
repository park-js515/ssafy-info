# 조회  

> DB 함수 - COUNT / DISTINCT  

집계함수 : count
distnct : 중복을 처리함  

```sql
select count(*) from emp; -- 괄호 안에 컬럼며을 넣는다.

select count(job) from emp; -- 중복 포함해서 갯수 세기
select count(distinct job) from emp; -- 중복 제거 후 갯수 세기  
```
