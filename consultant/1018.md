# 모바일 개발에 필수인 라이브러리, 프레임워크

### 라이브러리
Retrofit2: 서버와의 통신을 도와주는 라이브러리
Glide: 이미지를 손쉽게 처리하는 있도록 도와주는 라이브러리
Gson: Json <-> 객체

> 라이브러리의 장점
- 효율적인 코드 작성
- 손쉬운 버전 관리
- 효율적인 유지보수


### Jetpack
Android Jetpack
- Jetpack은 개발자가 중요한 코드에 집중할 수 있도록 권장사항, 준수, 사용구 코드 감소, 여러 Android 버전과 기기에서 일관되게 작동하는 코드 작성을 돕는 라이브러리 묶음
